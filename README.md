# News Classification using LLM (Qwen3-4B)

This repository contains a **news classification pipeline powered by a Large Language Model (LLM)** using the **Qwen3-4B model**.
The system classifies news articles based on their **title and content**, and evaluates the prediction accuracy against the ground-truth labels provided in the dataset.

# Dataset

The dataset is loaded from HuggingFace and the **test split** is used for evaluation.

Total number of news articles used for testing:

```
6459 news articles
```

Each record in the dataset contains the following fields:

```
[
 'id',
 'source',
 'title',
 'image',
 'url',
 'content',
 'date',
 'embedding',
 'created_at',
 'updated_at',
 'summary',
 'nomic_embedding',
 'label'
]
```

The **`label` field represents the ground truth category**, which is used to evaluate the classification results produced by the LLM.

# News Classification Process

The labeling process is performed **per news article** using an LLM.

### Steps

1. The dataset is downloaded from HuggingFace.
2. The **test split** containing 6459 news articles is loaded.
3. For each article:

   * A **context** is constructed using only:

     * `title`
     * `content`
4. The context is inserted into a **prompt template** located at:

```
source/prompts/agent_system_message_general.json
```

5. The prompt is sent to the **Qwen3-4B LLM model**.

6. The model generates a predicted news category.

7. The prediction is stored together with:

   * `id`
   * `true label`
   * `predicted label`
   * `response_id`

8. Results are saved to:

```
source/result/output_label.json
```

# Running the Pipeline

To run the full news classification pipeline:

```bash
python source/news_labeling_pipeline.py
```

This script will:

1. Load configuration from `config.conf`
2. Download the dataset
3. Run asynchronous LLM classification
4. Save predictions to `source/result/output_label.json`

# Notes

* Classification is performed **per article using an LLM inference call**.

* Accuracy may vary depending on:

  * prompt design
  * LLM model
  * dataset distribution.
